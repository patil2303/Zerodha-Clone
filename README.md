# Zerodha Clone

A full-stack clone of the Zerodha trading platform.

## 🔧 Tech Stack
- React
- Node.js
- Express
- MongoDB

## 🚀 Features
- Stock watchlist
- Holdings and positions simulation
- Responsive UI
- Clean dashboard design

## 🛠️ Installation
1. Clone the repo
2. Install dependencies in `/dashboard` and `/backend`
3. Run both servers
